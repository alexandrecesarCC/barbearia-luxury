# Barbearia Luxury

Este projeto é um template de site one‑page para barbearia de luxo desenvolvido por **Alexandre César**. Ele utiliza HTML5, CSS3 (incluindo design responsivo e glassmorphism) e JavaScript para criar uma experiência elegante.

## Seções
- Hero com vídeo e história fictícia;
- Nossa Equipe;
- Serviços;
- Depoimentos;
- Produtos;
- Planos VIP;
- Serviço de Agendamento;
- Rodapé com newsletter e mapa.

## Tecnologias
HTML, CSS, JavaScript vanilla. Não foram usados frameworks ou bibliotecas externas.
