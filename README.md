# **Barbearia Luxury** 

:globe_with_meridians: [Visite a versão hospedada](https://alexandrecesarcc.github.io/barbearia-luxury/#home)

Este projeto é um template de site one‑page para barbearia de luxo desenvolvido por Alexandre César. Ele utiliza HTML5, CSS3 (incluindo design responsivo e glassmorphism) e JavaScript puro para criar uma experiência elegante e interativa.

## **Seções**

» Hero: vídeo em preto‑e‑branco com efeito glass, logomarca e uma história fictícia sobre a barbearia;

» Nossa Equipe: perfis dos barbeiros com fotos e descrições;

» Serviços: cards animados com descrição de cortes e tratamentos;

» Depoimentos: feedbacks de clientes satisfeitos;

» Produtos: destaque para produtos de cuidados masculinos;

» Planos VIP: opções de assinatura e benefícios;

» Serviço de Agendamento: formulário de reservas com calendário e horários disponíveis;

» Rodapé: newsletter, contatos, links de navegação e mapa de localização.

## **Tecnologias**

O template foi construído com HTML, CSS e JavaScript sem frameworks ou bibliotecas externas. As folhas de estilo incluem variáveis CSS, flexbox, grid, animações e efeitos de glassmorphism. O JavaScript controla a navegação suave, modais de equipe e serviços, agendamento e interação do chatbot.

## **Como executar localmente**

### **Windows**

1. Faça o download ou clone este repositório para sua máquina.

2. Abra a pasta do projeto no Visual Studio Code ou outro editor.

3. O jeito mais rápido de visualizar é abrir o arquivo index.html diretamente no navegador (Chrome, Edge).

4. Para uma experiência de desenvolvimento melhor, instale a extensão Live Server e clique em “Open with Live Server”. Isso irá servir o site em http://localhost:5500 com atualização automática.

> [!NOTE]
> Alternativamente, abra o terminal no diretório do projeto e rode: python -m http.server 8000

### **Linux**

1. Baixe ou clone o repositório.

2. Abra o arquivo index.html diretamente no seu navegador.

3. Para iniciar um servidor local, abra o terminal na pasta do projeto e execute: python3 -m http.server 8000
> [!NOTE]
> Acesse então http://localhost:8000 no navegador. Você também pode usar servidores como live‑server do Node.js (npx live-server).

## **Idiomas**

Todo o conteúdo está originalmente em Português. Para adaptar o template para Inglês ou outro idioma:

1. Altere o atributo lang da tag <html> no index.html de pt-BR para o código do idioma desejado (por exemplo, en para inglês).

2. Traduza os textos das seções (títulos, parágrafos, botões) diretamente no HTML.

3. Caso deseje manter versões multilíngues, você pode duplicar a estrutura do site em pastas como /en/ e /pt/ e ajustar as traduções em cada uma.
